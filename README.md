Bearcnc is a complete-rewrite of [airbnb/javascript](https://github.com/airbnb/javascript) to support changes especially [ESLint V9](https://eslint.org/blog/2024/04/eslint-v9.0.0-released/). It still follows the [Airbnb Guideline](https://airbnb.io/javascript/).

**IMPORTANT** Backward compatibility is not supported until 1.0.0.
